# 온새나로 스튜디오

온새나로 스튜디오는 다양한 영역의 소프트웨어를 발하는 **오픈소스 중심 개발 스튜디오**입니다.
---

## 🚀 Goals

* 실사용 가능한 프로젝트 개발
* 깔끔하고 유지보수 가능한 구조 지향
* 오픈소스를 통한 협업 및 성장
* 특정 플랫폼에 종속되지 않는 개발

---

## 📦 Developing Approach

* **단순함 유지**: 불필요한 복잡도 지양
* **확장 가능성**: 구조적으로 성장 가능한 설계
* **실용성 우선**: 보여주기용보다 실제 사용성
* **지속 가능성**: 유지보수 가능한 코드

---
